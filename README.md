Chii is a chobit :)
##介绍
	唧 Chii 是动漫 人型电脑天使心 中的女主角
	使用PyQt编写QPainter加载图片,实现桌面拖动等
	使用 tuling123.com 的 openapi 实现对话功能
	使用psutil实现查看cpu使用率 (这个功能放在这里好像有点奇怪)
	持续开发中
##Required 
* Python2.7
* PyQt4
* psutil
